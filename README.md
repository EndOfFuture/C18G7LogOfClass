# C18G7LogOfClass
记录初一（18）班的奖罚情况。
# 规则
+ 奖  
  对于连续2周及以上没有扣分的学生，在班会上进行表扬。
+ 罚  
  + 对于被扣20分以上的学生，进行以下惩罚之一。
  + 全班公开检讨
  + 朗读班规、班级公约等
